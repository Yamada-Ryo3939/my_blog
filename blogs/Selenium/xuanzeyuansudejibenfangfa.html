Part 3.选择元素的基本方法

Yves2025/9/23技术学习Selenium自动化

其实关于这部分学过html、css等前端开发的同学应该都知道怎么回事，所以只需要粗浅看一下就知道怎么实现自动化了。

网页里一般存在着很多元素，比如说输入框、按钮等等，举个例子，如果进入某个购物网站，想查找某个商品名，人工的方式就是在输入框中输入想查找的商品的名字，再按下查询按钮。那么自动化要如何实现？

web界面自动化要操控元素，首先需要选择界面元素，或者说定位界面元素，也就是要告诉浏览器你要操作哪个界面元素，让它找到你要操作的界面元素，再进行下一步。下面介绍选择元素的基本方法。

选择元素的方法

选择元素要先告诉浏览器你需要操作的元素，就像叫某个人的名字，或者描述某个人的特征，元素也有这样的特征。

打开浏览器的开发者工具栏可以查看web元素，按下F12即可，点击Element，也就是下方图片的元素，即可查看页面对应的HTML元素（汉化方法如下图）：

然后点击最左侧的小箭头图标，将鼠标光标移到网站上的某个元素，就可以在下方的元素栏处查看它对应的html标签了。

根据id属性选择元素

打开某个网站，对着你想要查看的元素右键，选择**“检查”**，就可以在元素栏位看到高亮的代码，比如下图，当然，不是每个网页元素都有id的，如果查看的没有id是很正常的事情。

我们可以把id想象成元素的编号，也可以理解成身份证，因为根据规范，如果元素有id属性，这个id必须是当前html中唯一的。所以如果元素有id就用id选择元素，这样最为高效。

如何用自动化的驱使某个元素呢。假设有一个输入框，它存在id，那么往这个输入框里输入内容的代码示例如下（这里放我在网上查到的练习网站）：

from selenium import webdriver
from selenium.webdriver.common.by import By

options = webdriver.EdgeOptions()
options.add_experimental_option(
  'excludeSwitches',['enable-logging']
)

wd = webdriver.Edge(options = options)
wd.get('https://www.byhy.net/cdn2/files/selenium/stock1.html')

# 根据id选择元素，需要导包。返回的就是该元素对应的webelement对象，selenium3和4的写法不一样了，注意版本
element = wd.find_element(By.ID,'kw')

# 绑定完毕之后就可以操作元素，打回车的意思相当于点击了按钮，不打回车就不会开始查询
element.send_keys('通讯\n')

input('按回车退出')

wd.quit()

其中的

wd.find_element(By.ID,'kw')

使用了WebDriver对象的方法find_element，它的效果是发起一个请求通过浏览器驱动转发给浏览器，告诉它，需要选择一个id为kw的元素；浏览器找到之后就把结果通过浏览器驱动返回给自动化程序，这行代码就会返回一个WebElement 类型的对象。

然后我们就能操控这个对象了，比如：

  • 调用这个对象的 send_keys 方法就可以在对应的元素中输入字符串
  • 调用这个对象的 click 方法就可以点击该元素。

如果根据 传入的ID，找不到这样的元素，find_element 方法就会抛出 selenium.common.exceptions.NoSuchElementException 异常

根据class属性、tag名选择元素

根据class属性选择元素

除了根据元素的id，我们还可以根据元素的class属性选择元素。class属性代表了某种类型的元素，比如说茄子、土豆都是蔬菜，蔬菜就是所谓“class”。

这里推荐查看我练习的网站：白月黑羽测试网页1open in new window，点击f12按之前的操作就能查看：

所有的植物元素都有个class属性，值为plant。

所有的动物元素都有个class属性，值为animal。

如果我们要选择所有的动物，就可以这样写：

wd.find_elements(By.CLASS_NAME, 'animal')

注意element后面多了个s，因为这个class里包含了很多个元素，所以放在了一个列表中返回。如果不加这个s，那么就只会返回第一个元素。

要查看返回了哪些元素，可以用text属性查看：

from selenium import webdriver
from selenium.webdriver.common.by import By

options = webdriver.EdgeOptions()
options.add_experimental_option(
  'excludeSwitches',['enable-logging']
)

wd = webdriver.Edge(options = options)

wd.get('https://www.byhy.net/cdn2/files/selenium/sample1.html')

# 根据class选择元素，需要导包。返回的就是该元素对应的webelement对象
elements = wd.find_elements(By.CLASS_NAME,'animal')

# 取出列表中的每个 WebElement对象，打印出其text属性的值
# text属性就是该 WebElement对象对应的元素在网页中的文本内容
for elements in elements:
  print(elements.text)

input('按回车退出')

wd.quit()

有时候一个元素可以有多个class属性，比如：

<span class="chinese student">张三</span>

注意，这里 span元素 有两个class属性，分别 是 chinese 和 student， 而不是一个 名为 chinese student 的属性。

我们要用代码选择这个元素，可以指定任意一个class 属性值，都可以选择到这个元素，如下：

element = wd.find_elements(By.CLASS_NAME,'chinese')

✅或者：

element = wd.find_elements(By.CLASS_NAME,'student')

❌而不能这样：

element = wd.find_elements(By.CLASS_NAME,'chinese student')

根据tag名选择元素

跟用class选择元素一样，我们可以通过指定参数为 By.TAG_NAME ，选择所有的tag名为 div的元素，（tag其实就是尖括号里面的东西，比如页脚footer，按钮button）比如：

from selenium import webdriver
from selenium.webdriver.common.by import By

options = webdriver.EdgeOptions()
options.add_experimental_option(
  'excludeSwitches',['enable-logging']
)

wd = webdriver.Edge(options = options)

wd.get('https://www.byhy.net/cdn2/files/selenium/sample1.html')

# 根据 tag name 选择元素，返回的是 一个列表
# 里面 都是 tag 名为 div 的元素对应的 WebElement对象
elements = wd.find_elements(By.TAG_NAME,'div')

# 取出列表中的每个 WebElement对象，打印出其text属性的值
# text属性就是该 WebElement对象对应的元素在网页中的文本内容
for elements in elements:
  print(elements.text)

input('按回车退出')

find_element和find_elements的区别

使用 find_elements 选择的是符合条件的 所有 元素， 如果没有符合条件的元素， 返回空列表，也就是说它不会报错。

使用 find_element 选择的是符合条件的 第一个 元素， 如果没有符合条件的元素， 抛出 NoSuchElementException 异常。

通过WebElement对象选择元素

不仅 WebDriver对象有选择元素的方法， WebElement对象也有选择元素的方法。

WebElement对象 也可以调用 find_elements， find_element 之类的方法

WebDriver 对象 选择元素的范围是 整个 web页面， 而WebElement 对象 选择元素的范围是 该元素的内部。

拿之前的网站举例：

from selenium import webdriver
from selenium.webdriver.common.by import By

wd = webdriver.Chrome()

wd.get('https://www.byhy.net/cdn2/files/selenium/sample1.html')

element = wd.find_element(By.ID,'container')

# 限制 选择元素的范围是 id 为 container 元素的内部。
spans = element.find_elements(By.TAG_NAME, 'span')
for span in spans:
    print(span.text)

input('按回车退出')

输出结果只有：

内层11
内层12
内层21

相当于先在WebDriver对象中在整个页面筛选出id是container的元素，再在WebElement 对象中在上述筛选后的元素内部筛选出tag是span的元素。

通常的使用场景是没办法一步到位精确查找到某个元素的时候使用，多步跳转精确找到某元素。
